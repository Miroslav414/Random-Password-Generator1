## Описание программы

**Random Password Generator** – это графическое приложение на Python (Tkinter) для генерации надёжных паролей с настраиваемыми параметрами.  

Программа решает проблему создания сложных и уникальных паролей для различных сервисов, позволяет:
- выбирать длину пароля (от 4 до 30 символов);
- включать или исключать цифры, заглавные/строчные буквы и спецсимволы;
- копировать сгенерированный пароль в буфер обмена;
- сохранять историю всех паролей в JSON-файл для последующего просмотра.

Интерфейс интуитивно понятен, вся логика построена на стандартных библиотеках Python.
